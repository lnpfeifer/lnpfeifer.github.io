# Portfolio Website

A simple, minimalist single-page website to showcase my research projects and publications. Built with HTML, CSS, and JavaScript, designed for GitHub Pages with a clean black theme.

## About

This is a personal portfolio website featuring:
- Information about my background and research interests
- Publications and academic work
- Current and past projects
- Contact information via LinkedIn

## Tech Stack

- Pure HTML, CSS, and JavaScript
- No external dependencies or frameworks
- Responsive design for mobile and desktop
- Hosted on GitHub Pages
