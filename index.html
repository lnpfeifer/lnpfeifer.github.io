Preprints & Publications

Tool use fine-tunes arm and tool maps

Peviani V.C., Pfeifer, L.N., Risso, G., Bassolino M., Miller, L.E.

Radboud University, Donders Institute for Brain, Cognition and Behavior

There is evidence that the sensorimotor system builds fine-grained spatial maps of the limbs based on somatosensory signals. This study investigates whether a hand-held tool can be mapped in space with comparable spatial resolution and whether spatial maps change following tool use. Using a spatial mapping task on healthy participants, we measured the accuracy and precision of spatial estimates pertaining to several locations on their arm and on a hand-held tool. Results demonstrate that the spatial maps for the arm and tool were comparably accurate, suggesting that holding the tool provides enough information to the sensorimotor system to map it in space. Further, using the tool fine-tuned the user's spatial maps, increasing the precision of the tool map to a greater extent than their arm map, with increased precision focal to the tool tip, which may work as a spatial anchor following tool use.

Master Thesis - Neural Processing of a VR Horizon Task

Current Project

Raspberry Pi - Phoniebox Project

Past Project

This is a personal project with my father for the smallest members of our family. We used a Raspberry Pi Zero to build a "phoniebox". It is a music player that can be controlled via a web interface and RFID cards. The software allows Spotify integration, and can play music from local files as well. We used a Raspberry Pi Zero WH, RFID reader, and a two-speaker system. Of course, we upgraded the system with a sound card (HifiBerry).
